How To Play

How To Play

Ikuti langkah berikut untuk setup host di Android menggunakan Powertunnel.

Android Guide (Powertunnel)

  1. 1 - First, download Powertunnel APP from here.
  2. 2 - and then click on the "Powertunnel" button.
  3. 3 - It will copy the powertunnel host link.
  4. 4 - Then open the powertunnel app, click the three-line icon, and click on the host settings icon.
  5. 5 - Paste the copied powertunnel host URL. Also, change the host update period from 12 hours to on start.
  6. 6 - Now you can log in to your GTPS!
1
Install PowerTunnel

Download from official releases and install the APK.

2
Configure Host Settings

Open PowerTunnel → menu (≡) → Host Settings → Host list URL

3
Paste URL

Click button below and paste into PowerTunnel.

4
Start

Set Update period to On start, then press Start.

5
Launch Growtopia

Open Growtopia and click Play.
